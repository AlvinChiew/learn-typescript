# Learn TypeScript

## Getting Started

1. `npm start`
    - if error, abort and run again one more time

## Setup

1. npm init -y: To initialize project
1. npm packages:
    - Nodemon: Enable hotfix
    - Concurrently: To build and run tsc in one line
        - refer `package.json` scripts
    - Typings:
        - @types/node

1. tsc --init: To create tscconfig
    - "outDir": "./build"
    - "rootDir": "./src"
