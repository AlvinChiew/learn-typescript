# Learn TypeScript

## Quick Start

1. `ts-node index.ts`
    - compile & run
    - which consists of:
        1. `tsc index.ts`
            - compile tsc to js
        1. `node index.js`
            - run js

## Installation

1. Typescript: `npm install -g typescript ts-node`
1. Axios (network request): `npm install axios`
